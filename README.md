# キャッシュフロー管理アプリ

個人と法人（法人）のキャッシュフローを予実管理するWindowsデスクトップアプリです。

## 初回セットアップ

1. **Pythonをインストール**（まだの場合）
   - https://www.python.org/downloads/ からPython 3.11以上をダウンロード
   - インストール時に **「Add Python to PATH」** にチェックを入れる

2. **`setup.bat` をダブルクリック**
   - 必要なライブラリ（PySide6）が自動インストールされます

3. **`run.bat` をダブルクリック**
   - アプリが起動します

## 初回の使い方

### まず設定（⚙️ マスタ設定タブ）

1. **サービスマスタ** → 法人のサービス種別を登録（サービスA、コンサルなど）
2. **固定費マスタ** → 個人・法人の毎月の固定費を登録（食費、電気代、SaaSなど）
3. **目標設定** → 個人・法人の月次目標を設定

### 毎月の操作

1. **月初** → 今月の状況タブを確認。固定費・積立が自動展開されます
2. **収入が入ったとき** → 個人/法人タブで「入金確認 ✓」ボタンを押す
3. **請求書が来たとき** → 光熱費・食費の実績額を入力
4. **15日・月末** → 予定外支出をまとめて入力

## データの保存場所

`C:\Users\（ユーザー名）\cashflow_app\data.db`

SQLiteファイルです。バックアップする場合はこのファイルをコピーしてください。

## ファイル構成

```
cashflow-app/
├── main.py              アプリ起動ファイル
├── database.py          データベース操作
├── style.py             デザイン設定
├── requirements.txt     必要なライブラリ
├── setup.bat            初回セットアップ（ダブルクリック）
├── run.bat              アプリ起動（ダブルクリック）
└── ui/
    ├── tab_dashboard.py     今月の状況タブ
    ├── tab_budget.py        個人・法人 予実タブ
    ├── tab_subscriptions.py サブスク顧客タブ
    ├── tab_accounts.py      口座残高・積立タブ
    ├── tab_memo.py          備忘・TODOタブ
    └── tab_settings.py      マスタ設定タブ
```
